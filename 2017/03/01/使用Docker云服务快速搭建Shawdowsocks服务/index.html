使用Docker云服务快速搭建Shawdowsocks服务

使用Shawdowsocks作为proxy工具，shawdowsocks有各个平台的客户端，可以在PC和手机上使用（如何使用shawdowsocks 请自行谷歌学习）。

使用Docker云服务商（这里选用 时速云 ）搭建docker版本的Shawdowsocks server，

登录注册充值一系列流程之后，

  • 进入应用管理，选择创建应用
  • 选择 镜像仓库，下一步
  • 搜索镜像shadowsocks，然后找到排名靠前的shadowsocks-server 选择部署
  • 在下一步的配置中填入对应的名称等信息，在高级设置中设置自己的Password，选择创建
  • 然后在应用列表中启动服务，然后就能看的应用的访问地址。
  • 回到PC或者手机上的Shawdowsocks客户端，填入上一步的应用访问地址和设置的Password，设置完成。

打开shawdowsocks客户端，然后在浏览器中查看自己的IP，已经变成服务器对于的地址的IP，OK 配置结束。

时速云是按照小时收费，不用服务的时候可以关闭时速云的应用。

同理如果选择国外的docker云服务商。
